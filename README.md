# She Can Foundation Contact Management System

A modern full-stack contact form web application built for the She Can Foundation Internship Task.

## Features

* Responsive Modern UI
* Contact Form Submission
* MongoDB Database Integration
* REST API using Express.js
* Admin Dashboard
* Form Validation
* Mobile Friendly Design
* Full Stack Web Application

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js
* MongoDB Atlas

## Project Structure

